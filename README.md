## Descripción

Sistema de Contactos es una aplicación web desarrollada con Spring Boot que permite administrar una lista de contactos de manera sencilla e intuitiva.
La aplicación implementa operaciones CRUD (Crear, Ver, Editar y Eliminar) utilizando Spring MVC para el controlador,
Thymeleaf para las vistas y Spring Data JPA para la persistencia en PostgreSQL.

## Características Principales

- Gestión completa de contactos con operaciones CRUD
- Interfaz web amigable con plantillas Thymeleaf
- Persistencia de datos con PostgreSQL
- Validación de formularios
- Arquitectura MVC (Modelo-Vista-Controlador)
